# LexiPath

LexiPath is a focused Windows English-learning application built in Rust.

The first course path is fixed and non-configurable:

1. IPA foundation
2. Ogden Basic English 850
3. Oxford 3000
4. Controlled reading paired with every vocabulary unit

Core rules:

- A lesson only passes after every required item is answered correctly.
- Wrong answers are retrained until the remaining error set reaches zero.
- Every vocabulary unit includes controlled reading.
- Learner-facing English in a lesson is validated against the cumulative learned-word whitelist.
- English words, example sentences, and readings can be played aloud.
- Course content is data-driven so later official stages can be added without changing the learning engine.

The repository is currently being initialized.
